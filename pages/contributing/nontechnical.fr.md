## Je veux aider, mais je ne code pas !

Nous comprenons que le codage n'est pas pour tout le monde, mais il y a encore beaucoup de choses avec lesquelles vous pouvez nous aider, même si vous ne codez pas/ne pouvez pas/ne voulez pas coder :D 
<br>

**Ajouter des agences :**
Vous remarquez qu'il y a une agence que nous n'avons pas ? Faites-nous savoir ce qui nous manque sur Github Issues ! Allez sur ce [lien](https://github.com/catenarytransit/vehicles/issues) et vous devriez voir l'écran suivant :

![github issues page](../../examples/new_issue.png)

Appuyez sur le bouton vert qui dit "New issue" près du coin supérieur droit de l'écran, sous la barre supérieure.

Si vous n'avez pas de compte github, veuillez vous inscrire !
Si vous en avez un, vous pouvez remplir le formulaire avec l'agence que vous souhaitez ajouter.

Pour vous aider, nous avons écrit un modèle ! Copiez-collez simplement les informations suivantes dans chaque champ et remplacez les informations entre crochets par ce que vous savez. Ce n'est pas grave si vous n'avez pas toutes ces informations non plus, nous pouvons aider à les trouver !

- Title: Add data for [Nom complet de l'agence] aka [Nom abrégé ou court de l'agence]
- Add a description: Hey! I would like to request data for the agency mentioned above. This agency serves the [région] area and you can find more information at [site officiel, ou, encore mieux, API pour leurs données !].

Cette image montre un exemple de ce à quoi cela peut ressembler :
![github issues page](../../examples/issue.png)

Après avoir rempli cela, appuyez simplement sur le bouton vert avec le mot "Create" dessus et c'est bon !

Une dernière note : Toutes les agences ne publient pas leurs données et toutes les données ne sont pas standardisées pour notre système. Nous ferons de notre mieux pour acquérir les informations pour mettre à jour Catenary, mais nous ne pouvons pas dire avec certitude que nous pouvons tout ajouter ce dont vous avez besoin :pensive:
<br>
