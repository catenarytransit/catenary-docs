# Flux de travail

Introduisons-nous au flux de travail de Catenary. Assurez-vous de lire la page GitHub si vous êtes nouveau sur GitHub !

Comme indiqué précédemment, Catenary utilise GitHub pour gérer notre base de code ; c'est aussi ce qui nous rend open source — n'importe qui peut voir et copier notre code ! Pour ajouter à notre branche officielle cependant, nous demandons aux contributeurs de faire des pull requests via GitHub comme moyen de promouvoir la communication et les améliorations pour notre base de code opérationnelle. (Remarque : si vous gravissez les échelons en tant que contributeur régulier cependant, nous pouvons vous accorder des permissions de push vers main !)

Si vous n'avez pas rejoint notre Discord, il est fortement suggéré de le faire, car c'est une ligne directe vers les autres développeurs ! Nos Guides sont très actifs sur notre serveur et répondent rapidement. Les Guides sont des mainteneurs qui ont travaillé en étroite collaboration avec certains projets Catenary et sont facilement disponibles pour aider les gens. Si vous faites une Pull Request, postez dans les discussions sur GitHub, ou pinguez **@Guides** sur Discord, ce sont eux qui répondront ! En octobre 2024, ces Guides sont :

Catenary n'exige ni n'a de réunions régulières comme les clubs scolaires ou les organisations professionnelles ; nous essayons de garder tout asynchrone puisque nous avons des membres issus d'une grande variété d'horizons et de fuseaux horaires. Nous *avons* des événements de type assemblée publique où nous passons en revue les sujets pertinents pour notre développement et nos activités, et des équipes plus petites peuvent décider de programmer des discussions également ! De plus, si vous voyez quelqu'un dans nos canaux vocaux Discord, n'hésitez pas à rejoindre et à écouter et/ou parler (respectueusement, bien sûr) — nous sommes heureux d'expliquer sur quoi nous travaillons ou simplement de discuter avec notre communauté !

Vous pouvez également essayer de nous contacter sur nos divers comptes de réseaux sociaux. Si vous êtes ici, il y a de fortes chances que vous nous ayez trouvés via l'un d'eux !
